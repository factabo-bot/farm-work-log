# farm-work-log

研修先農場の作業記録・報告システム。LINEグループのチャット報告を、タップ式の構造化記録に置き換える。

- 計画書: Obsidian `00_Inbox/研修先_作業記録システム_計画書.md`
- セットアップ: [docs/セットアップ手順.md](docs/セットアップ手順.md)

## 構成

```
frontend/   LIFFアプリ（素のHTML/JS、ビルド不要）
  index.html   パート用の記録画面
  app.js       画面ロジック（GAS未設定ならお試しモードで動作）
  masters.js   棟・列・作業のマスタ（現地調査後に差し替え）
  config.js    LIFF ID / GAS URLの設定
gas/
  Code.gs      バックエンド（記録受付・1マス1行展開・スタッフ自動登録）
docs/
  セットアップ手順.md
```

## 現状（v0）

- パート用記録画面: ①棟選択 → ②配置図タップ（複数マス可）→ ③作業選択（その他は記述欄）→ 複数作業をまとめて記録
- 今日記録済みのマスはグレー表示
- GAS未設定でもブラウザ単体で動作確認できる（localStorage保存）
